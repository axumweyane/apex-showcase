5 Safety Guardrails trading/safety/

Signal Variance Guard
Halts pipeline if signal scores collapse (std < 0.01). Prevents undiversified random bets.
Halts pipeline
Leverage Gate
Skips order batch if portfolio leverage exceeds 1.5x. Continues to snapshot/reporting.
Skips orders
Calibration Health
Verifies model calibration at startup. Logs error and skips calibration if drift detected.
Logs warning
Model Promotion Gate
Rejects models with Sharpe ratio below 0.5 from going live. Prevents poisoned ensemble signals.
Rejects model
Execution Failure Monitor
Pauses remaining orders if failure rate exceeds 25% within the execution window.
Pauses batch

17-Step Daily Pipeline

1
Fetch OHLCV via yfinance (stocks + SPY + FX pairs)
2
Circuit breaker check (Redis-backed drawdown limits)
3
Detect market regime (calm/volatile x trending/choppy)
4
Run 12 strategies (momentum, pairs, FX, options, sentiment)
5
Bayesian ensemble combine (60% Sharpe + 40% regime weights)
6
Signal variance check — halt if scores collapse
7
Risk assessment (VaR/CVaR, correlation gates, kill switches)
8
Portfolio optimization with position/leverage constraints
9
Leverage gate — skip if leverage > 1.5x
10
Execution failure monitor — pause if failure rate spikes
11
Execute via Alpaca (market/limit orders)
12
Audit trail logging to PostgreSQL
13
PostgreSQL persistence (trades, snapshots, signals)
14
Feed risk manager with daily return
15
Redis pub/sub signal distribution
16
Discord + Email reports via NotificationManager
17
LLM signal analysis via Ollama (pattern detection)
